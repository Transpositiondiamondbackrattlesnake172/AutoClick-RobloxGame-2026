# Universal Auto Clicker for Roblox | Performance & Accessibility Tool
A high-performance, lightweight automation utility designed to simulate mouse clicks within the Roblox environment. This tool is built to enhance user experience by automating repetitive tasks and reducing physical strain.

# 📌 What is it?
The **Roblox Auto Clicker** is a productivity enhancement tool that automates mouse interactions. It is designed for users who need to perform consistent, rapid, or long-term clicking tasks without manual input, ensuring a smoother and more efficient workflow.

# 🚀 Why use an Automation Tool?
Many experiences within the Roblox ecosystem involve repetitive mechanics. Using an automation utility provides several benefits:

Hardware Protection: Significantly reduces wear and tear on your physical mouse switches.

- Ergonomics: Prevents repetitive strain injury (RSI) and hand fatigue during extended sessions.

- Efficiency: Optimizes progression in simulators and idle-based games by maintaining consistent interaction rates.

# 🎮 Ideal for Various Genres
This utility is perfectly suited for:

- Simulators: For collecting resources or training character stats.

- Clicker Games: Where progression depends on interaction frequency.

- Idle Experiences: To prevent being flagged as inactive and maintain presence in the session.

# ✨ Key Features
- Customizable Intervals: Set delay times from milliseconds to hours.

- Click Selection: Supports Left, Right, and Middle mouse button emulation.

- Click Type: Toggle between single and double-click modes.

- Infinite or Timed Loops: Run until manually stopped or for a specific number of repetitions.

- Global Hotkeys: Seamlessly start and stop the utility with customizable keyboard shortcuts (Default: F6).
